📌 Problem Statement

In hospitals, delays in communicating medicine and supply requests from patient rooms to the pharmacy can reduce efficiency, especially during emergencies.
Many small and medium hospitals lack automated systems to handle this process effectively.

💡 Proposed Solution

We propose a low-cost autonomous robot using Arduino that can:

Navigate through hospital paths
Stop at each room
Detect presence/absence of requests
Count total requests accurately
Travel to the pharmacy station
Report data using LED and buzzer

⚙️ System Features
🚗 Path following robot
🛑 Room detection and stopping mechanism
📡 Request detection system (sensor/button based)
🔢 Request counting logic
💡 LED indication (number of requests)
🔊 Buzzer feedback (number of rooms)

🧠 System Architecture 
Robot follows predefined path by line following 
Stops at each room
Detects request input
Updates request counter
Moves to next room
After completing all rooms → reaches pharmacy
Outputs result via LED & buzzer
